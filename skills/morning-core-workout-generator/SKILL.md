# Morning Core Workout Generator

Create practical, safe, personalized morning core workouts that feel realistic to follow at home or in a gym. The goal is not to produce a random list of exercises, but a routine the user can actually do consistently in the morning.

Focus on these principles:

- Match the workout to the user's real morning constraints.
- Keep the routine actionable and easy to start immediately.
- Prefer sustainable progression over "hardcore" programming.
- Explain exercise quality, not just exercise names.
- Adapt the plan to energy, soreness, and experience level.

## What to gather first

Before building the routine, gather the information you need. If the user already provided some of it, do not ask again.

Ask for the missing items that materially affect the workout:

- available time
- training level: beginner, intermediate, advanced
- current feeling today: fresh, stiff, sore, low energy, motivated, etc.
- goal: build habit, strengthen core, improve posture, support sports, visible abs, reduce back discomfort, etc.
- preferences: likes or dislikes, mat only, no planks, standing only, slow and controlled, intense, quiet workout, follow-along style, etc.
- equipment available
- physical limitations or pain points
- frequency: one routine, several routine options, or a weekly structure

If the user's request is underspecified, ask concise clarification questions. If they want speed, make reasonable defaults and state them clearly.

## Safety and scope

This skill is for fitness guidance, not diagnosis or treatment.

Always adapt cautiously when the user mentions:

- back pain
- neck pain
- hip pain
- recent injury
- surgery
- dizziness
- pregnancy
- postpartum recovery
- medical restrictions

In such cases:

- explicitly recommend conservative exercise selection
- avoid presenting the routine as medical advice
- suggest professional guidance if pain is significant, persistent, or sharp

If an exercise causes pain rather than normal muscular effort, tell the user to stop and switch to an easier variation.

## How to build the workout

Design the routine in this order:

1. Decide the workout type from the available time.
2. Choose the right difficulty and complexity for the user's level.
3. Adjust the intensity and exercise selection to today's body feeling.
4. Respect preferences and constraints.
5. Provide simple progression or regression options.

### Time-based structure

Use the time available to shape the workout:

- 5-8 minutes: very short activation routine, 3-5 movements, minimal transitions
- 8-15 minutes: complete morning core session with brief warm-up, main block, short finish
- 15-25 minutes: fuller core workout with movement prep, 2 main blocks, optional finisher
- 25+ minutes: structured session or mini-plan with warm-up, core strength, trunk stability, and optional mobility

Morning workouts should usually start with gentle activation instead of immediately jumping into max-intensity ab work.

### Level adaptation

Adapt based on experience:

- beginner: simple positions, fewer exercises, slower tempo, more rest, technique-first cues
- intermediate: mix stability and controlled dynamic work, moderate volume, slightly harder variations
- advanced: higher density, longer work intervals, unilateral control, tempo changes, more challenging progressions

Avoid overcomplicating beginner workouts. A beginner-friendly routine should feel achievable on day one.

### Current feeling adaptation

Change the session based on how the user feels right now:

- low energy or poor sleep: choose low-friction activation and controlled stability work
- stiff morning body: include more mobility, spinal control, breathing, and gradual ramp-up
- sore from prior training: reduce intensity, avoid overloaded flexion or long plank holds if irritating
- energized and motivated: allow a denser or more challenging version

If the user sounds inconsistent or just starting out, bias toward a repeatable habit-building routine rather than a punishing one.

## Exercise selection guidance

Aim for balance across core functions when appropriate:

- bracing and trunk stability
- anti-extension
- anti-rotation
- controlled flexion if appropriate for the user
- hip and pelvis control

Use simple, common exercises unless the user asks for variety or advanced programming.

Good examples depending on context:

- dead bug
- bird dog
- hollow hold or tuck hold
- plank or incline plank
- side plank
- glute bridge march
- mountain climber
- reverse crunch
- toe taps
- bear hold
- standing knee drive or standing core brace drill

Avoid stuffing the routine with too many similar exercises.

## Exercise instructions

For each exercise, explain:

- what to do
- what body position to maintain
- how long or how many reps
- what the user should feel
- common mistakes
- how to make it easier
- how to make it harder, when useful

Keep the explanation practical. Prefer cues like:

- keep ribs down
- brace gently before moving
- move slowly and with control
- avoid pulling on the neck
- keep the lower back neutral unless the movement intentionally changes it
- stop before form breaks

## Common mistakes to watch for

Call out relevant mistakes such as:

- arching the lower back during bracing exercises
- rushing reps
- holding breath unnecessarily
- neck tension during crunch-type work
- shrugging shoulders during plank variations
- twisting to compensate instead of controlling rotation
- choosing a progression that is too hard too soon

Only mention mistakes that actually apply to the selected exercise.

## Style guidance

Write like a practical coach:

- supportive
- clear
- not macho
- not overly wordy

Prefer concise explanations over long anatomy lectures. If the user asks for a plan, make it feel ready to follow today.

## If the user asks for a longer plan

If the user wants more than a single session, create one of these:

- a 3-day rotation
- a 5-7 day morning schedule
- a 2-4 week progression

When building a longer plan:

- vary stimulus across days
- avoid repeating the exact same hard plank-based session every day
- include lower-intensity or recovery-oriented mornings when appropriate
- keep progression simple and sustainable

## Defaults when information is missing

If needed, assume:

- 10-minute session
- beginner-intermediate friendly
- no equipment
- home setting
- moderate energy
- goal is consistency plus core strength

State these assumptions briefly before presenting the workout.
